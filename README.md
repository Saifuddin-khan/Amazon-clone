# 🛒 Amazon Clone

A responsive Amazon homepage clone built using **HTML5** and **CSS3**, designed to replicate the look and feel of Amazon's user interface with a clean, modern, and responsive layout.

---

## 🚀 Features

- Responsive Homepage Layout
- Amazon-inspired Navigation Bar
- Hero Banner Section
- Product Cards
- Multiple Shopping Sections
- Professional Footer
- Clean and Organized Code

---

## 🛠️ Technologies Used

- HTML5
- CSS3
