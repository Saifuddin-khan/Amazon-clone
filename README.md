# 🛒 Amazon Clone

A fully responsive **Amazon Homepage Clone** built using **HTML5** and **CSS3**. This project replicates Amazon's homepage layout with a modern, clean, and user-friendly design while following a well-organized project structure.

1. Clone the repository

```bash

```

2. Open the project folder.

3. Run `index.html` in your browser.
